![api](https://user-images.githubusercontent.com/50766393/127772947-fd9020f3-7781-4d00-baba-f3f0d655b7c2.png)

  * Navigation

  * Room

* UI
  * ConstraintLayout
  * RecyclerView
  * Fragment
  * Activity
  * Service
  * Notification
  * BottomNavigationView
  * Fab

  * Dialog

* Third Party
